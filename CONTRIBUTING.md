# Contributing Guidelines / 기여 가이드라인

RheeWorks ForConvinience 프로젝트에 기여해 주셔서 감사합니다! 
Thank you for contributing to the RheeWorks ForConvinience project!

## Rules / 규칙

1. **Bilingual Documentation**: All new features and documentation must be bilingual (Korean and English).
   - 모든 새로운 기능과 문서는 반드시 한영병기여야 합니다.
2. **Premium Aesthetics**: UIs must follow the "Premium Aesthetics" guidelines (HSL colors, CustomTkinter, modern fonts).
   - UI는 반드시 "프리미엄 미학" 가이드라인을 따라야 합니다.
3. **Commit Messages**: Follow the `Korean / English` format.
   - 커밋 메시지는 `한글 / 영어` 형식을 따릅니다.
4. **License**: Apache 2.0 must be maintained.
   - Apache 2.0 라이선스를 유지해야 합니다.
